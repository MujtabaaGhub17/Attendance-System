# Attendance-System
# 🎓 Attendance System using Computer Vision (CN Project)
## 📌 Overview
This project is an **Automated Attendance System** built using **Computer Vision techniques**.
Instead of manual attendance, the system detects and recognizes faces to mark attendance automatically.

It was developed as part of a **Computer Networks (CN) mini project**, integrating concepts of automation and real-time data handling.

---

## 🚀 Features

* 📷 Real-time face detection using webcam
* 🧠 Face recognition for identifying individuals
* 📝 Automatic attendance marking
* 📊 Attendance stored in CSV format
* ⚡ Fast and efficient processing

---

## 🛠️ Technologies Used

* Python
* OpenCV (Computer Vision)
* NumPy
* Pandas
* Face Recognition libraries

---

## ⚙️ How It Works

1. The system captures live video using a webcam
2. Detects faces in the frame
3. Matches detected faces with stored data
4. Marks attendance with timestamp
5. Saves attendance in a CSV file

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/Attendance-System.git
```

2. Navigate to the project folder:

```
cd Attendance-System
```

3. Install dependencies:

```
pip install opencv-python numpy pandas
```

4. Run the project:

```
python main.py
```
